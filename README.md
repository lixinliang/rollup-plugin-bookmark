# rollup-plugin-bookmark
> Rollup plugin to bundle js as bookmark.

## Installation
```
npm install --save-dev rollup-plugin-bookmark
```

## Usage
```js
import { rollup } from 'rollup';
import bookmark from 'rollup-plugin-bookmark';

rollup({
    entry: 'main.js',
    plugins: [
        ookmark()
    ]
}).then(...)
```
